# API Reference

This app exposes some demo endpoints:

- `GET /` → Hello message
- `GET /health` → JSON health check
- `GET /items` → List of items
- `GET /items/:id` → Get item by id
- `GET /search?q=term` → Search echo
- `POST /echo` → Echo back JSON payload
